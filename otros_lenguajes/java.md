# Java

## Sintaxis

Java es un lenguaje compilado, tipado estáticamente y orientado a objetos. Su sintaxis exige declarar tipos, organizar el código en clases y usar un método `main` como punto de entrada en aplicaciones sencillas.

Temas iniciales:

- Variables y tipos primitivos.
- Operadores.
- Condicionales.
- Bucles.
- Métodos.
- Paquetes e importaciones.

## POO fuerte

Java se apoya intensamente en la programación orientada a objetos. El diseño suele organizarse en clases con responsabilidades claras y relaciones explícitas.

Conceptos clave:

- Clases y objetos.
- Encapsulación.
- Herencia.
- Polimorfismo.
- Interfaces.
- Clases abstractas.

## Colecciones

El framework de colecciones proporciona estructuras listas para almacenar, buscar y recorrer datos de forma eficiente.

Colecciones frecuentes:

- `ArrayList`.
- `LinkedList`.
- `HashMap`.
- `HashSet`.
- `TreeMap`.
- `Queue` y `Deque`.

## Excepciones

Las excepciones permiten representar y manejar errores durante la ejecución del programa. Java distingue entre excepciones verificadas y no verificadas.

Buenas prácticas:

- Capturar solo errores que se puedan manejar.
- Crear excepciones propias cuando aporten claridad.
- Usar `finally` o try-with-resources para liberar recursos.
- Evitar ocultar errores importantes.

## Spring Boot básico

Spring Boot simplifica la creación de aplicaciones Java, especialmente APIs y servicios web. Proporciona configuración automática, servidor embebido y una estructura común para proyectos empresariales.

Primeros temas:

- Controladores REST.
- Servicios.
- Inyección de dependencias.
- Configuración con `application.properties` o `application.yml`.
- Acceso a datos con Spring Data JPA.
- Pruebas básicas de endpoints.
