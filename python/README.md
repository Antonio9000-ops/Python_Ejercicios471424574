# Reglas de dependencias por tema

Cada archivo de ejercicios solo puede usar temas ya estudiados. La idea es practicar un concepto a la vez y evitar adelantar recursos que todavía no pertenecen al nivel correspondiente.

## Reglas generales

- Cada archivo solo puede usar temas ya estudiados en archivos anteriores o en el tema actual.
- En `05_bucles.py` se pueden usar variables, entrada/salida, operadores, condicionales y bucles.
- No usar listas hasta `07_listas.py`.
- No usar funciones hasta que se llegue al tema de funciones.
- No usar archivos, módulos, POO, APIs ni bases de datos en niveles básicos.

## Tabla de dependencias por tema

| Archivo | Tema principal | Conceptos permitidos |
| --- | --- | --- |
| `01_variables.py` | Variables | Variables, tipos básicos y asignación. |
| `02_entrada_salida.py` | Entrada/salida | Variables, tipos básicos, asignación, `print()` e `input()`. |
| `03_operadores.py` | Operadores | Variables, entrada/salida, operadores aritméticos, comparación y lógicos. |
| `04_condicionales.py` | Condicionales | Variables, entrada/salida, operadores, `if`, `elif` y `else`. |
| `05_bucles.py` | Bucles | Variables, entrada/salida, operadores, condicionales, `while` y `for`. |
| `06_strings.py` | Cadenas de texto | Variables, entrada/salida, operadores, condicionales, bucles y operaciones básicas con strings. |
| `07_listas.py` | Listas | Variables, entrada/salida, operadores, condicionales, bucles, strings y listas. |
| `08_funciones.py` | Funciones | Variables, entrada/salida, operadores, condicionales, bucles, strings, listas y funciones. |
| `09_diccionarios.py` | Diccionarios | Todos los conceptos anteriores y diccionarios. |
| `10_archivos.py` | Archivos | Todos los conceptos anteriores y lectura/escritura de archivos. |
| `11_modulos.py` | Módulos | Todos los conceptos anteriores y uso básico de módulos. |
| `12_poo.py` | Programación orientada a objetos | Todos los conceptos anteriores, clases y objetos. |
| `13_apis.py` | APIs | Todos los conceptos anteriores y consumo básico de APIs. |
| `14_bases_datos.py` | Bases de datos | Todos los conceptos anteriores y operaciones básicas con bases de datos. |

Si un ejercicio necesita un concepto que todavía no aparece en esta tabla, debe simplificarse o moverse al archivo del tema correspondiente.
