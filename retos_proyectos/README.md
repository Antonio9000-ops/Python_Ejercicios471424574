# Retos y proyectos de Python

Esta carpeta reúne retos prácticos organizados por dificultad para acompañar el aprendizaje progresivo de Python. Cada archivo propone proyectos adecuados al nivel del estudiante e incluye objetivo, requisitos, restricciones, extras opcionales y criterios de evaluación.

## Cómo usar estos retos

1. Elige el archivo correspondiente a tu nivel actual.
2. Lee el reto completo antes de programar.
3. Implementa primero los requisitos mínimos.
4. Añade extras opcionales solo cuando la versión base funcione correctamente.
5. Evalúa tu solución con los criterios indicados.

## Niveles disponibles

| Nivel | Archivo | Enfoque |

| Principiante | [`principiante.md`](principiante.md) | Entrada/salida, variables, condicionales y ciclos básicos. |
| Intermedio | [`intermedio.md`](intermedio.md) | Listas, diccionarios, funciones, archivos y organización del código. |
| Avanzado | [`avanzado.md`](avanzado.md) | APIs, persistencia, validaciones, manejo de errores y estructura de proyectos. |
| Profesional | [`profesional.md`](profesional.md) | Base de datos, autenticación, tests, despliegue y buenas prácticas de producción. |

## Recomendaciones generales

- Escribe código claro y con nombres descriptivos.
- Divide el problema en pasos pequeños.
- Prueba manualmente cada funcionalidad antes de avanzar.
- Evita copiar soluciones completas: usa documentación y ejemplos para entender.
- Guarda versiones de tu trabajo para comparar mejoras.
