# Proyectos y retos de portafolio

Esta carpeta contiene desafíos diseñados para convertir conceptos de Python en evidencia práctica. Cada reto debe resolverse respetando el nivel de conocimientos alcanzado y evitando herramientas que todavía no correspondan.

## Cómo trabajar un reto

1. Lee el enunciado completo.
2. Escribe los requisitos mínimos en una lista de tareas.
3. Implementa una primera versión simple.
4. Prueba manualmente con varios datos.
5. Refactoriza nombres y elimina duplicación.
6. Documenta uso, decisiones y próximos pasos.
7. Agrega tests cuando el nivel ya incluya funciones y módulos.

## Retos sugeridos por nivel

### Nivel 1: básico

- Juego de adivinanza por terminal.
- Calculadora de propinas.
- Conversor de unidades.
- Validador de contraseña simple.
- Simulador de cajero con saldo fijo.

Restricciones: variables, entrada/salida, operadores, condicionales, bucles y strings básicos.

### Nivel 2: intermedio

- Gestor de tareas en memoria.
- Agenda de contactos.
- Analizador de texto.
- Registro de gastos simple.

Restricciones: listas, diccionarios, funciones, archivos y validaciones.

### Nivel 3: avanzado

- CLI organizada en módulos.
- Consumo de API pública.
- Automatizador de archivos.
- Proyecto con logging y manejo de errores.

### Nivel 4 y 5: profesional

- API REST con tests.
- Aplicación con base de datos SQL.
- Dashboard de datos.
- Proyecto full-stack integrando Python, TypeScript y SQL.

## Plantilla recomendada para cada proyecto

```text
nombre_del_proyecto/
├── README.md
├── src/
├── tests/
└── docs/
```

Incluye siempre: objetivo, instalación, ejecución, ejemplos, decisiones técnicas y mejoras futuras.
